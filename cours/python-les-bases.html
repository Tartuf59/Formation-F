Apprendre Python

Introduction

Installer Python :

Étape 1 : Télécharge Python

Va sur le site officiel de Python : https://www.python.org/downloads/.

Clique sur le bouton Download Python [version] (la version sera probablement la dernière version stable, comme 3.x.x).

Une fois le fichier téléchargé, ouvre-le pour lancer l'installation.

Étape 2 : Installe Python

Important : Avant de cliquer sur "Install Now", coche la case "Add Python.exe to PATH". Cela te permettra d'utiliser Python facilement depuis n'importe où.

Clique sur "Install Now" et laisse l’installation se terminer.

Une fois l’installation terminée, clique sur Close.

Étape 3 : Vérifie l’installation

Ouvre un terminal (ou invite de commande) :

  • Sur Windows : Appuie sur Win + R, tape cmd, puis appuie sur Entrée.
  • Sur Mac/Linux : Ouvre le Terminal depuis le menu.

Tape la commande suivante et appuie sur Entrée :

python --version
ou
python3 --version

Si tu vois une version comme Python 3.13.1, Python est bien installé !

Étape 4 : Test rapide

Toujours dans le terminal, tape la commande suivante :

python

Cela ouvrira l'interface interactive de Python. Tu devrais voir >>, ce qui signifie que Python est prêt. Essaie d'écrire :

print("Hello, world!")

Si tu vois Hello, world!, tu es prêt !

Astuce : Installer et configurer Python avec Sublime Text

Sublime Text est un éditeur de texte léger et très populaire pour coder. Voici comment l'utiliser pour programmer en Python.

Étape 1 : Télécharge Sublime Text

Va sur le site officiel de Sublime Text : https://www.sublimetext.com/.

Clique sur "Download" et choisis la version adaptée à ton système d’exploitation (Windows, macOS ou Linux).

Étape 2 : Installe Sublime Text

Lance le fichier téléchargé pour démarrer l’installation. Suis les étapes de l’assistant d’installation. Une fois terminé, ouvre Sublime Text.

Étape 3 : Configure Sublime Text pour Python

Ajoute Python à Sublime Text :

  1. Dans Sublime Text, ouvre le menu Tools > Build System > New Build System.
  2. Un fichier s'ouvre. Remplace son contenu par :
{
  "cmd": ["python3", "-u", "$file"],
  "file_regex": "^[ ]*File \"(...*?)\", line ([0-9]*)",
  "selector": "source.python"
}

Enregistre le fichier avec le nom Python3.sublime-build.

Choisis Python comme système de build : Dans le menu Tools > Build System, sélectionne Python.

Étape 4 : Teste ton installation

  1. Dans Sublime Text, crée un nouveau fichier : File > New File.
  2. Enregistre-le avec une extension .py (par exemple, test.py).
  3. Écris le code suivant :
print("Bienvenue dans Python avec Sublime Text!") # "print sert à écrire dans le résultat"
# Ceci est un commentaire qui n’affecte pas le code, on le commande avec un "#"

Appuie sur Ctrl + B (ou Cmd + B sur macOS) pour exécuter ton programme.

Si tout est bien configuré, le message apparaîtra dans le panneau de sortie en bas.

Étape 5 : Installe des bibliothèques Python

Si tu veux utiliser des bibliothèques supplémentaires comme math, pandas ou numpy, utilise pip (le gestionnaire de paquets Python).

Ouvre un terminal ou une invite de commande.

Tape :

pip install nom_de_la_bibliothèque

Par exemple, pour installer pandas :

pip install pandas

Exemple : Utiliser une bibliothèque en Python

Python propose une bibliothèque standard incluse par défaut.

Exemple simple avec math :

import math # Importation de la bibliothèque math
# Utilisation d'une fonction de la bibliothèque
resultat = math.sqrt(16) # Calcul de la racine carrée de 16
print(resultat) # Affiche : 4.0

Exécute ce code dans Sublime Text pour vérifier que tout fonctionne.

Bibliothèques populaires en Python

  • Manipulation des données :
    • pandas : Travailler avec des tableaux de données.
    • numpy : Manipuler des ensembles de données et faire des calculs mathématiques avancés.
  • Graphiques :
    • matplotlib et seaborn : Pour tracer des graphiques.
  • Développement web :
    • flask et django : Pour créer des applications web.
  • Jeux :
    • pygame : Pour créer des jeux simples en 2D.
