Introduction à Python

Bienvenue dans cette formation qui te permettra d'apprendre les bases du langage Python, l’un des langages les plus populaires et faciles à prendre en main.

Tu peux la télécharger ici ou la suivre juste en dessous.

Ce que tu vas apprendre

  • Installer Python et vérifier son installation
  • Configurer Sublime Text pour coder en Python
  • Utiliser pip pour installer des bibliothèques Python
  • Comprendre les variables et types de données
  • Utiliser les conditions (if, elif, else)
  • Maîtriser les boucles (for, while)
  • Créer et utiliser des fonctions
  • Manipuler les listes
  • Utiliser des bibliothèques standards (math, random)
  • Manipuler les fichiers (lecture, écriture, ajout)
  • Exécuter et tester des programmes complets en Python

Débutant – Découverte des bases en programmation avec Python

Ouvre Sublime Text :

  • Lance Sublime Text.
  • Ouvre un nouveau fichier et sauvegarde-le avec l'extension .py (par exemple : premier_programme.py).

Prêt ? Maintenant, écris ce code simple dans ton fichier :

# Afficher un message
print("Bonjour, le monde !")

Explication :

  • print : Fonction qui affiche du texte ou des infos à l'écran.
  • Le texte entre guillemets est le message à afficher.

Un programme de calcul :

# Faire un calcul simple
print(5 + 3)
print(10 * 2)
print(15 / 3)

Explication :

  • + : Addition
  • * : Multiplication
  • / : Division

es variables et les types de données

nom = "Alice"
age = 25
taille = 1.68
is_student = True

print("Nom :", nom)
print("Âge :", age)
print("Taille :", taille)
print("Student :", is_student)

Types de données :

  • int : entier, ex. 42
  • float : nombre décimal, ex. 3.14
  • str : chaîne de caractères, ex. "Bonjour"
  • bool : booléen, ex. True ou False

Exercice interactif

Crée une variable pour :

  • Ton prénom
  • Ton âge
  • Une réponse booléenne si tu aimes Python

Puis affiche ces informations.

Solution :

# Déclaration des variables
mon_prenom = "Tom"
mon_age = 20
j_aime_python = True

# Affichage des informations
print("Prénom :", mon_prenom)
print("Âge :", mon_age)
print("Aime Python :", j_aime_python)
